Interactive Help

Use help(arg) or ?arg

To get help on a command, use ?cmd or help("cmd"); e.g. 

bsh % ?print
bsh % help("print");
bsh % help("dnorm");

To get help on a BEASTObject, use help(BEASTObject); e.g. 

bsh %  help(beast.util.MCMC);
bsh %  var = new beast.util.MCMC();
bsh %  ?var

To get help on any other class, use help(Class); e.g. 

bsh %  help(java.util.ArrayList);
bsh %  data = new ArrayList<Double>();
bsh %  ?data
