BEAST Shell getting started

Running demos

You probably want to start with the demos first, which show off the capabilities of BEAST Shell. To list available demos, run 

bsh % demo();
which lists available demos. To run one of the demos, add it as arrugment to demo() for instance, to run the chart demo, run 
bsh % demo("chart");

Help

To get help on a command, use bsh % help("cmd"); e.g. 

bsh % help("print");
bsh % help("dnorm");

To get help on a BEASTObject, use bsh % help(BEASTObject); e.g. 

bsh %  help(beast.util.MCMC);

To get help on any other class, use bsh % help(Class); e.g. 

bsh %  help(java.util.ArrayList);
