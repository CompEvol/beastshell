BEAST Shell getting started

Running demos

You probably want to start with the demos first, which show off the capabilities of BEASTShell. To list available demos, run 

bsh % demo();
which lists available demos. To run one of the demos, add it as arrugment to demo() for instance, to run the chart demo, run 
bsh % demo("chart");

Interactive Help

To get help on a command, use help("cmd"); of alternatively ?cmd e.g. 

bsh % help("print");
bsh % ?print
bsh % help("dnorm");
bsh % ?dnorm

To get help on a BEASTObject, use help(BEASTObject); e.g. 

bsh %  ?beast.core.MCMC
or, if you already created an MCMC object, asking help for the object will have the same results: 
bsh % tree = new beast.evolution.tree.Tree()
bsh % ?tree
bsh % javap(tree)
bsh % ??tree

To get help on any other class, use help(Class); e.g. 

bsh %  help(java.util.ArrayList);
bsh %  ??java.util.ArrayList;
bsh %  data = new ArrayList<Double>();
bsh %  ?data
