"regexp" documentation

Description:

regexp(str, regexp) matches a regular expression and returns an array of matches.

Usage:

	matches = regexp(str, regexp);
	matches[0]; // matches complete regular expression
	matches[1]; // matches first group
	matches[2]; // matches second group
...
If there are multiple matches, several blocks will be returned: 
	s = "dog doggie";
	matches = regexp(s, "dog");
	// matches is now [dog, dog]

	matches = regexp(s, "(d)(og)");
	// matches is now [dog, d, og, dog, d, og]
