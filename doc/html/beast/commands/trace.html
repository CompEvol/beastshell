"trace" documentation

Description:

trace(filename, burnin, variable) shows trace plot of variable variable with burnin burnin of a tracefile with name filename. Also, shows statistics of the trace with the given burn-in.

Usage:

trace(filename, burnin, variable);

e.g.

	trace("anolis.log");
plots "posterior" from "anolis.log" with 10% burn-in 
	trace("../data/anolis.log", 20);
plots "posterior" from "../data/anolis.log" with 20% burn-in 
	trace("primates.log", 20, "likelihood");
plots "likelihood" from "primates.log" with 20% burn-in

Arguments:

filename: name of the tracefile.

burnin: burn-in percentage, default 10

variable: variable name in the trace file to plot, default is "posterior"
