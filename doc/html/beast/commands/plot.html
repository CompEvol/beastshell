"plot" documentation

Description:

plot(component), plot(filename) -- create a plot or export to file.

Usage:

plot(component);

plot(filename);

e.g.

 plot(treeDrawing);
      plot("tree.png");
      plot("tree.pdf");
      plot("tree.tex");

Arguments:

Component component: JComponent to be plotted

String filename: file to save the component to, the file extension is used to determine the file format. Use one of png, bmp, jpg, pdf, tex. Output to tex produces a tix drawing in a latex file.
