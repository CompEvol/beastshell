"help" documentation

Description:

help is the primary interface to the help systems.

Usage:

?command

?class

?variable

help("command")

help(class)

help(variable)

Arguments:

command: show help for a command

class or variable: show help for class (or class of the variable) if it is a BEASTObject otherwise show javadoc information for the class from oracle.
