"open" documentation

Description:

open(filename) open a file for reading, writing or appending

Usage:

open("beast.xml") -- open file "beast.xml" for reading. Returns a BufferedReader so the file can be ready line by line using 
while (fin.ready()) {
	s = fin.readline();	
}

open(">beast.xml") -- open file "beast.xml" for writing. Returns a PrintStream so can be used with print and println, like so 
out = open(">beast.xml");
out.println("<beast version='2.2'/>");
out.close();

open(">>beast.xml") -- open file "beast.xml" for appending. Also returns a PrintStream.
