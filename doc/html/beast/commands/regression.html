"regression" documentation

Description:

regression(x, y) calculates regression line with coordinates (x[i], r[i]) where r[i] are the points prediceted by a linear regression approximation of the points (x[i], y[i]).

Usage:

regression(x, y);

e.g.

 r = regression(x, y)

Arguments:

x: array or collection of Numbers.

y: array or collection of Numbers.

Returns:

array of double.
