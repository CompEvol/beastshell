Product of Vector Elements

Description

prod returns the product of all the values present in its arguments.

Usage

prod(x)

Arguments

x

Array or collection of numeric values.

Details

If a Doble.NaN value in any of the arguments will cause a value of Double.NaN to be returned.

Value

The product, a numeric (of type "double"). NB: the product of an empty set is one, by definition.

See Also

sum, cumprod, cumsum.
