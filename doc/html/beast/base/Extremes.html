Maxima and Minima

Description

Returns the maxima and minima of the input values.

Usage

max(x), max(x, na_rm = false)
min(x), min(x, na_rm = false)

Arguments

x

Array or collection of numeric values.

na_rm

boolean. If true NaN values are removed before the result is computed.

Details

Value

returns a double representing the minimum, or maximum of the input x. Throws exceptions when x is empty;

See Also

range (both min and max)
