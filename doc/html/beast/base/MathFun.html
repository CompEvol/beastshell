Miscellaneous Mathematical Functions

Description

abs(x) computes the absolute value of x, sqrt(x) computes the (principal) square root of x, √{x}.

The naming follows the standard for computer languages such as C or Fortran.

Usage

abs(x)
sqrt(x)

Arguments

x

A single numeric value or array or collection of numeric values.

Details

abs(x) returns an integer vector when x is integer

See Also

Log for logarithmic, sin for trigonometric, and Special for special mathematical functions.
