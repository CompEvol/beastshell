Cumulative Sums, Products, and Extremes

Description

Returns a vector whose elements are the cumulative sums, products, minima or maxima of the elements of the argument.

Usage

cumsum(x)
cumprod(x)
cummax(x)
cummin(x)

Arguments

x

an array of int, double or Number or List of Number.

Value

An array of the same length and type as x (after coercion).

An NaN value in x causes the corresponding and following elements of the return value to be NaN, as does integer overflow in cumsum.
