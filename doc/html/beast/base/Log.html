Logarithms and Exponentials

Description

log computes logarithms, by default natural logarithms, log10 computes common (i.e., base 10) logarithms, and log2 computes binary (i.e., base 2) logarithms. The general form log(x, base) computes logarithms with base base.

log1p(x) computes log(1+x) accurately also for |x| << 1.

exp computes the exponential function.

expm1(x) computes exp(x) - 1 accurately also for |x| << 1.

Usage

log(x)
log10(x)
log2(x)

log1p(x)

exp(x)
expm1(x)

Arguments

x

A single numeric value or array or collection of numeric values.

Details

Value

A number or List of the same length as x containing the transformed values. log(0) gives Double.NEGATIVE_INFINITY, and log(x) for negative values of x is Double.NaN. exp(-Inf) is 0.
