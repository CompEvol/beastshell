BEAST Studio -- a basic IDE for BEASTShell

The panels

  • BEASTShell console
  • BEASTShell editor
  • Variable inspector
  • History panel
  • Help pane
  • Plot panel

Some handy short-cut keys

+/- for zooming out and in of the help pane
Ctrl-G for search/replace in the editor pane
Ctrl-F for search in the help and editor pane as well as filtering the history

Accessing BEASTStudio from BEASTShell

To print to the console, use the print command, or bsh.console.print(). The latter can be used to set the font as well, e.g 
bsh.console.print("Hello world", new Font("SansSerif", Font.BOLD, 12));
this.interpreter.studio is the BEASTStudio object, which can be used to add panels to the chart-panel, or open web-pages, or display information in HTML format in the help panel. 
// set text in Help panel to "hello world"
this.interpreter.studio.helpPane.setText("hello world");

// display beast2.org in help panel
this.interpreter.studio.helpPane.setURL(new URL("http://beast2.org"));
To add a component to the chart-panel, call addChart on the plotPane. The following adds a button to the plotPane, but you can imagine adding some nice graphics to the plotPane. 
button = new JButton( "My Button" );
this.interpreter.studio.plotPane.addChart(button);
The plotPane allows export to bitmap or PDF, so it allows you to draw some things, then save them to file. Also, the plotPane allows browsing through the components that were added.
