BEAST Studio variables

All variables present in the console are listed in alphebtical order in the variables panel.

By clicking a variable, its contents is shown in the text entry at the bottom, where you can copy the value to clipboard by clicking in the textfield so it gets the focus, then using Ctrl-A (to select all text in the entry) followed by Ctrl-C (to copy the selection to the clipboard).

History is automatically saved when closing BEASTStudio, and loaded when starting BEASTStudio again.
