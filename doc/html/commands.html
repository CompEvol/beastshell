BEASTShell Commands

Commands are divided into
  • BEAST commands which are mostly specific to BEAST.
  • BeanShell commands which are inherited from BeanShell. Some commands are deleted or adjusted for BEASTShell.
  • Math, statistical and utility functions for general purpose usage.

BEAST commands

Some useful commands are
demodemonstrate some BEASTShell capabilities in BEASTStudio
helpprovide online help
cconstruct a list of objecst, e.g. x=c(1,1.5,2,2.4)
assertEqualshandy for testing, e.g. assertEquals(dbeta(0.4, 5, 1, false), 0.128, 0.0001);
beaststart a BEAST run
beautistart a BEAUti session
appstorestart the BEAST AppStore
requiresindicate some BEAST package is required, and install if not present.
